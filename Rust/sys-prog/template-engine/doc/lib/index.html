ExpressionData

Enums

ContentType

ContentType is the main data structure to classify the template string read from the template file. It is represented as enum and contains the list of possible token types read from the template file. As each statement (template string) is read from the template file, it is evaluated to check if it is one of the types defined in this enum.

TagType

TagType is a supporting data structure that is used to indicate whether a template string corresponds to a for-tag (repetitive loop) or if-tag (display control)
